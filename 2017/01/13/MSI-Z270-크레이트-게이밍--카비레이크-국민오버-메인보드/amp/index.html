MSI Z270 크레이트 게이밍

인텔 7세대 카비레이크 발표와 함께 200 시리즈 메인보드들이 대거 출시되었습니다. B250 / H270 / Z270 칩셋을 사용한 다양한 제품들이 출시되었고, 그 중에서도 i7-7700K 오버클럭킹과 함께 Z270 메인보드 들이 관심을 받고 있습니다.

다양한 제품 속에서 가장 많은 변화를 보여준 메인보드 하나를 살펴보려 합니다. 지난 Z170 출시와 함께 전압 강하로 큰 수난을 겪었던 Z170 크레이트 게이밍의 후속작 MSI Z270 크레이트 게이밍이 바로 그 주인공 입니다.

지금 부터 제품 스펙 / 외형과 함께 바이오스 구성을 함께 소개하며, 추가적으로 인텔 코어 7세대 카비레이크 i7-7700K를 장착하여 7700K@4.8GHz , 7700K@5.0GHz 오버클럭킹 가이드를 소개합니다.

제품 스펙

제품 소개

패키지는 제품 이름(KRAIT = 독사)에서 알 수 있듯이 뱀 비늘 디자인을 보여주며, 기존에 뱀 비늘을 파괴하는 발톱 자국이 추가된 모습을 보여줍니다. 무언가 새로운 모습을 보여주려는 의지가 패키지에서 확인됩니다.

뒷면에는 제품 외형과 주요 기능들에 대하여 표기하고 있습니다.

패키지를 개봉하면 Z270 크레이트 게이밍과 구성품이 담겨져 있습니다.

제품 설명서와 소프트웨어 DVD를 포함한 다양한 구성품이 함께 제공됩니다.

Z270 크레이트 게이밍 외형은 기존 Z170에서 보여줬던 블랙 & 화이트 컨셉을 이어가면서도 후면 I/O 쉴드, 그리고 CPU 전원부 방열판에서 부터 칩셋 방열판까지 발톱에 긁혀 속살을 보여주는 듯한 패키지와 제품까지 동일한 컨셉의 디자인을 채택하고 있습니다. 블랙 & 화이트 컨셉의 제품 중에서 가장 일관성 있고 멋진 형태의 디자인이라 생각됩니다.

이번 시리즈에서 추가된 후면 I/O 쉴드는 CPU 전원부 방열판과 함께 알루미늄 소재가 채택되어 기능적 / 디자인적으로 우수한 모습을 보여주고 있습니다.

CPU 주변으로 12페이즈 전원부가 위치하고 있으며 다크 쵸크, 다크 캐패시터를 사용하여 낮은 전력에서 안정성과 뛰어난 수명을 제공합니다.

폭스콘 1151 소켓이 사용되어 인텔 7세대 전제품을 지원합니다. ( 인텔 7세대 , G / i3 / i5 / i7 지원 )

비슷한 가격대에서 가장 많은 6개의 팬 커넥터를 제공하여 이를 통해 CPU 팬을 비롯한 다수의 시스템 팬을 바이오스와 MSI APP을 통해 제어 할 수 있습니다. 6개의 커넥터 중에는 수냉 전용 워터펌프 커넥터가 제공되어 수냉 쿨러를 위한 최적화가 가능합니다.

MSI 스틸아머 기술이 적용된 M.2 소켓을 2개 제공하며 일반 SATA III의 5배가 넘는 빠른 전송이 가능합니다. 이 소켓을 통해 M.2 2242 ~ 22110 까지 장착이 가능하며 INTEL OPTANE 메모리도 연결할 수 있습니다. 또한, M.2 Genie란 기능을 통해 손쉽게 RAID 구성이 가능하며 이를 통해 3배 이상의 성능 향상을 기대할 수 있습니다.

우측 하단에는 SATA III 6Gb/s 포트 6개와 USB 3.0 헤더가 확인됩니다.

이전 시리즈부터 적용되었던 PCI-E 스틸아머는 기판 디자인에 어울리도록 Hydro Dip Paint로 처리가 되어있습니다.

메인 칩셋 Z270 에는 MSI 게이밍 로고가 그려진 메탈 방열판이 부착되어 있습니다.

하드웨어 모니터링을 위한 nuvoTon 센서가 확인됩니다.

내장 오디오 AUDIO BOOST 4 칩셋이 적용되어 수준급의 사운드를 제공합니다.

테스트 시스템

인텔 코어 7세대 카비레이크 i7-7700K (정품). ( 16년 39주차 / 기본 전압 1.168v )

G.SKILL DDR4 16G PC4-25600 CL16 TRIDENT ZKY (8Gx2) x 1EA.

G.SKILL DDR4 16G PC4-25600 CL16 TRIDENT Z (8Gx2) x 1EA. ( Total 32G / XMP 3,200MHz 16-18-18-38 )

SCYTHE FUMA 대양케이스. ( with MX-4 )

UEFI BIOS

바이오스 첫 화면은 EZ MODE로 시작하며 마우스를 이용하여 손쉽게 제어할 수 있습니다.

CPU / Memory / Storage / Fan Info 등으로 핵심 요소를 제어할 수 있습니다.

M-Flash라는 기능을 통해 USB로 메인보드 바이오스를 손쉽게 갱신할 수 있습니다.

하드웨어 모니터에서는 CPU / PUMP / FAN 1 ~ 4 까지 6가지 팬을 제어할 수 있으며 PWM / DC 모드로 자동 제어, 또는 메뉴얼 제어가 가능합니다.

좌측 상단에 GAME BOOST 버튼을 통해 프리셋을 통핸 CPU OC이 가능합니다. 오버클럭킹이 어려운 초심자를 위해 추천하는 기능이지만 오버클럭킹 가이드를 본 리뷰 내에 제공할 예정으로 직접 오버클럭킹 하는 것을 권장합니다.

좀 더 상세한 설정이 필요한 경우 EZ MODE 에서 F7을 눌러 ADVANCED MODE로 전환할 수 있습니다.

Settings 메뉴를 통해 시스템 정보, 부팅 설정 및 Z270 칩셋이 제공하는 다양한 기능들을 제어할 수 있습니다.

본 제품의 핵심 메뉴 중 하나로 OC 메뉴를 통해 CPU / Memory 오버클럭킹을 설정할 수 있으며 상세한 기능을 심플하게 제공합니다. 해당 메뉴에 대한 상세한 설명은 오버클럭킹 가이드 항목에서 상세히 소개하도록 하겠습니다.

오버클럭킹 가이드

셋팅값 설명에 앞서 수정이 필요한 메뉴에 대한 설명을 진행합니다.

가장 먼저 OC Explore Mode [ Normal ] → [ Expert ] 로 설정하여 상세메뉴를 활성화 시킵니다.

CPU Ratio Mode는 모든 코어를 오버클럭킹 할 수 있는 [ All Core ]로 설정합니다. 상황에 따라 코어 당 각개 설정이 필요한 경우 Per Core를 사용할 수는 있으나 일반적으로 All Core를 권장합니다.

Extreme Memory Profile = X.M.P 메뉴의 경우 일반 메모리에서 사용하지 않으며, XMP 프로파일을 갖춘 OC 메모리를 사용할 경우 해당 메뉴를 통해 Profile을 선택하여 설정을 불러올 수 있습니다.

예) XMP 메모리 : G.Skill, Kingston, Avexir, Corsair 등

DigiALL Power 메뉴에는 전압강하를 방지하기 위한 CPU Loadline Calibration Control ( LLC )를 설정할 수 있습니다. 설정값에 대한 효과는 우측에 그래프로 설명이 되어있으며 오버클럭킹 상황에서는 LLC Mode 4 ~ 5 사이를 추천합니다.

코어 전압을 제어하는 몇 가지 모드가 존재하며 일반적으로 [ Override Mode / Offset Mode ]를 사용합니다. 오버가이드에서는 좀 더 쉽게 접근할 수 있도록 Override Mode (Fixed 또는 Manual Mode)로 안내합니다.

오버클럭킹 확인

OC 설정 후 실사용이 가능한지 테스트가 필요합니다. 이러한 과정을 통해 각 CPU의 수율에 따라 전압을 줄이거나 늘리는 작업이 필요하며 이 과정을 오버클럭킹 안정화 과정이라고 합니다.

테스트 준비물 : CPU-Z v1.78.1 / Realtemp v3.7 / LinX v0.6.4 (AVX)

- CPU 정보를 보여주는 프로그램으로 Core Voltage / Speed / Muliplier 등을 실시간으로 체크할 수 있으며, 간단한 벤치마크 기능을 통해 현재 CPU의 Single / Multi Thread 성능을 수치로 확인할 수 있습니다.

* 카비레이크 한계 온도 : 100도 ( 초과시 다운클럭 발생 )

- CPU 코어의 온도를 실시간으로 표시, 그리고 온도 초과 상태를 기록해주는 프로그램으로 과부하 테스트와 함께 켜놓고 실제 최대 온도는 몇도인지를 체크하게 됩니다.

일반적으로 오버클럭킹 적용 후 100% 로드상황에서 최대 온도 90도 내외라면 온도는 높은 편이나 실사용이 가능한 수준이며, 최대 온도를 줄이기 위해서는 [ CPU 쿨러 업그레이드 ] , [ 써멀 재도포 ] , [ CPU 코어 전압 다이어트 ] 등의 방법이 있습니다. 이 중에서 가장 손쉬운 것이 CPU 코어 전압 조절로 입력된 전압값을 낮추어 온도를 떨어뜨릴 수 있습니다.

- CPU와 메모리를 테스트하는 프로그램으로 100% 로드 상황을 강제할 수 있습니다. 해당 스크린샷은 6,144MB로 설정하였으나 테스트 진행에는 실제 메모리를 기준으로 사이즈를 잡아야 합니다. 기준은 아래와 같습니다.

* 8GB : 6,144MB 권장

* 16GB : 12,288MB 권장

* 32GB : 24,576MB 권장

최신 버전의 LinX 0.7.0에 있는 AVX2 명령어의 경우 CPU 부하를 더 높게 잡기 때문에 이것을 통과한다면 보다 안정적인 오버클럭킹 적용상태라 할 수 있지만, 실제로 최신 LinX를 통과하기는 그 기준이 너무 높기에 실사용 수준인 0.6.4로 진행하여도 좋습니다. 0.6.4 를 20바퀴 통과한 후 전압을 살짝 올려서 사용하는 것을 권장드립니다.

프리징, 재부팅, 블루스크린 해결방안

앞서 소개한 오버클럭킹 확인 과정에서 화면이 멈추거나(프리징), 재부팅, 블루스크린과 같은 현상이 발생합니다. 이러한 현상이 발생하면 아래와 같은 조치를 취해야합니다.

- CPU 전압 추가 : 입력된 전압으로는 해당 배수를 유지할 수 없게되면 오작동이 발생합니다. 안정적인 작동을 위해 전압값을 추가하여 다시 테스트를 진행합니다. ( 0.010 ~ 0.020v 수준으로 추가하며 테스트)

- CPU 캐쉬 조절 : CPU 캐쉬값은 CPU 배수보다 2단계정도 낮아야합니다. CPU 배수와 같거나 2단계보다 높은 경우 오류가 발생할 수 있습니다. 만약, 2단계 아래에서 오류 발생시 CPU 배수를 낮추거나 캐쉬 배수를 한단계 더 낮추어야 합니다.

- 앞선 방법을 모두 진행하여도 오작동이 발생하는 경우 : 이 경우에는 안타깝지만 CPU 수율의 문제로 인해 높은 오버클럭킹은 적용하기 어렵습니다. 배수와 전압을 낮추어가면서 적정 온도에서 최대 클럭을 찾으시기 바랍니다.

지금까지 MSI Z270 크레이트 게이밍의 외형, 바이오스, 그리고 오버클럭킹에 대하여 살펴보았습니다.

간략하게 제품 평을 해보자면 최근 유행하는 화이트 튜닝에 걸맞는 블랙 & 화이트 구성을 갖추고 있고, 비슷한 가격대에서 제품 구성이 풍부하여 가성비가 우수한 편이 속합니다. ( 12페이즈 전원부, 6개의 핀 헤더, 알루미늄 히트싱크 등 )

눈여겨 볼 부분은 지난 시리즈에서 이슈가 되었던 전압 강하 문제인데, 이번 MSI Z270 제품군은 출시 초반부터 타사에 비해 완벽할 정도의 오버클럭킹 기능에 있어서 안정적인 모습을 보여주었습니다. 메뉴 구성이 상세하지만 접근하기 용이하며 강화된 LLC (CPU Loadline Calibration Control) 기능으로 칼전압을 제공하는 점이 매우 인상적입니다.

우수한 오버클럭킹 기능으로 인해 i7-7700K와 조합하여 국민오버 4.8GHz , 그리고 대망의 5.0GHz 까지 안정적으로 오버를 할 수 있었으며, 카비레이크 CPU의 발열만 제대로 제어할 수 있다면 그 보다 높은 클럭도 무리없이 소화할 수 메인보드라 생각됩니다.

제조사의 분골쇄신, 절치부심이 느껴지는 이번 MSI Z270 크레이트 게이밍은 카비레이크 국민오버를 위한 메인보드로 많은 오버 유저에게 꾸준한 사랑을 받게될 것이라 예상해봅니다.
